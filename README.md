# Year in the Life of a Manager

An interactive narrative experience by Valence showcasing key moments in a manager's journey throughout the year, powered by the Nadia platform.

## Overview

This is a single-page web application that walks through pivotal management moments — from team onboarding to organizational insights — illustrating how Nadia supports managers at each stage.

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no build step)
- Google Fonts (Playfair Display, DM Sans, Space Mono)

## Running Locally

Open `index.html` in a browser, or serve with any static file server:

```sh
npx serve .
```
